# 🚗 Road Accident Prediction Project

## 📌 Description
This project is a Flask-based web application that analyzes road accident data and provides insights/predictions.

## 🚀 Features
- Data analysis using Pandas
- Web interface using Flask
- Displays accident-related data

## 🛠 Tech Stack
- Python
- Flask
- Pandas
